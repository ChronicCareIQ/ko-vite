# ko-vite
Demo the use of VITE and modules

## Install
```
cd c:/dev/ko-vite/KoVite
npm install vite -D
```

## To run
```
cd c:/dev/ko-vite/KoVite
node_modules\.bin\vite.cmd
```
Then open the browser at http://localhost:5173
